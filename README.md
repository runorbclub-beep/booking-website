# 霸族预约助手 - 网页版

多语言预约平台，支持专家和场地管理。

## 🌐 访问地址

- **客户预约页面**：[booking.html](booking.html)
- **管理员后台**：[admin.html](admin.html)
- **场地浏览**：[browse.html](browse.html)
- **入驻申请**：[register.html](register.html)
- **超级管理员**：[super-admin.html](super-admin.html)

## 📝 功能说明

### 客户端
- 浏览可用场地和专家
- 选择日期和时段进行预约
- 查看预约记录

### 管理端
- 使用场地名称和管理密码登录
- 审核预约申请
- 管理白名单
- 生成小程序码和分享链接
- 修改管理密码

### 超级管理员
- 审核入驻申请
- 管理所有租户

## 🔧 技术栈

- 纯HTML/CSS/JavaScript
- 响应式设计
- 多语言支持（中文/英文）
- Cloudflare Workers后端API
